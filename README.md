# Respuesta al Revisor 2

Proyecto LaTeX independiente para preparar la carta de respuesta a las observaciones del Revisor 2.

Archivo principal:

```text
main.tex
```

Compilación sugerida:

```powershell
latexmk -pdf main.tex
```

El documento está organizado en tres columnas:

- ID de la observación.
- Observación del Revisor 2.
- Recuadro para documentar la respuesta, el cambio realizado y la ubicación en la tesis.
